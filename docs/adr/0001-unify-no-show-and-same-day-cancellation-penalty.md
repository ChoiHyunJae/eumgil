# 노쇼와 당일 취소를 단일 페널티 카운터로 합산

동행(escort) 생명주기에서 노쇼(약속 시간 30분 경과까지 "만났어요" 미확인)와 당일 취소(약속 당일 명시적 취소)는 실제로는 다른 사건이지만, 둘 다 "약속 위반"이라는 상위 개념으로 묶어 같은 페널티 카운터(누적 3회 시 7일 매칭 제한)에 합산하기로 결정했다. 별도 카운터로 분리하는 대안도 검토했으나(예: 당일 취소는 더 가벼운 패널티), 미용실 예약 패널티 모델을 그대로 차용해 규칙을 단순하게 유지하는 쪽을 택했다.

용어는 분리한다 — `노쇼`는 미확인 사건에만, `당일 취소`는 사전 취소 사건에만 쓰고, 카운터·패널티 로직은 둘 다 "약속 위반" 발생 시 동일하게 트리거된다. 나중에 당일 취소에 더 가벼운 패널티를 주고 싶다면 카운터를 분리해야 하며, 이는 기존 누적 데이터의 마이그레이션을 동반한다.
